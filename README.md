# LSK_TGC_Analizer
Две модели, одна выдает топ 3 категории канала, другая полное его описание, написан файлик для удобного использования и внедрения моделей в проекты. Модель с топ3 категории была дообучена на данных(за основу взята RuBERT), которые я самостоятельно собрал и предобработал. Вторая модель - интегрированная LLM.

Дообучение RuBert(файл texttoclass):
Система автоматической классификации Telegram-каналов по тематикам на основе анализа последних публикаций.
Технологии: Hugging Face Transformers, PyTorch, Telethon (Telegram API) для асинхронного сбора данных, Scikit-learn (Label Encoding), регулярные выражения для чистки текста
Модель анализирует не один пост, а совокупность текстов, выдавая топ-3 доминирующих категории канала. Реализован удобный пайплайн предсказания, от ввода юзернейма канала до получения финальной тематики.
